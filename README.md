# invoice-management
<h1>Angular 7 used</h1><br>
<b>Clone this Repo</b><br>
<b>cd invoice-management/</b><br>
<b>Run "npm install"</b><br>
  <b>Run "ng serve --open"</b>
<p>It will start listening on http://localhost:4200/</p>
